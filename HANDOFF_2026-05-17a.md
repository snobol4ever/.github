## Key Insight from This Session

The `lower_stmt` early-return guard is a recurring hazard: it fires when a node has no `:lbl`, `:stno`, `:subj`, `:pat`, `:eq`, or goto attrs — i.e., any bare `TT_*` node that isn't a `TT_STMT`. **Never call `lower_stmt(x)` when `x` is not a `TT_STMT` or one of the special-cased types at the top of `lower_stmt` (`TT_LOOP_BREAK`, `TT_LOOP_NEXT`, `TT_DEFINE`, `TT_END`).** For any other node kind arriving as a subject, call its dedicated `lower_*` function directly.
