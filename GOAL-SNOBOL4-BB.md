THE CONVENTION (the invariant): MODE 3 has exactly ONE C→BB transfer — the driver MAIN branch (scrip.c ~:1327, `m3_enter_with_rbx(fn, rt_frame(), 0, arena)` / `(void)fn(rt_frame(), 0)`).
