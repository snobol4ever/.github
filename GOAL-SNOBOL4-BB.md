STEPS (run to fixpoint; the full gate suite is the safety net that proves each removal behavior-neutral):

1. **ROOTS — the emitted runtime surface.** Compile the FULL corpus (every language; exercise every CLI switch — `--run`, `--compile`, `--target=`, `--dump-*`, `--trace`, `--bench`, `--monitor`, `--case-sensitive`) to `.s`. Scrape every emitted symbol reference (`call <sym>@PLT`, `lea`/`mov` of a named symbol) -> `ROOTS_EMIT` = the runtime functions emitted code ACTUALLY calls. mode-3's RX slab is emitted by the SAME templates (absolute `movabs rax,&sym; call rax`), so its call-set == `ROOTS_EMIT` by template symmetry — confirm by scraping the emitter call-table (`x86("call","sym",(uint64_t)(uintptr_t)sym)`).
2. **scrip binary — the linker GC oracle.** Rebuild scrip with `-ffunction-sections -fdata-sections` and link `-Wl,--gc-sections -Wl,--print-gc-sections 2>gc.log`. The linker PROVES (deterministically, from `main` + kept refs) every `.text.<fn>`/`.data.<obj>` it removes -> `DEAD_LINKER`. Address-taken dispatch targets are RETAINED by the linker (it sees the reloc), so `--gc-sections` is SAFE — it never drops a live indirect-call target.
3. **libscrip_rt.so — static call-graph reachability.** `DEFINED = nm -D --defined-only`. `ROOTS = ROOTS_EMIT ∪ exported-API ∪ driver hook/dlsym names`. Build the caller->callee edge set via `objdump -d`; BFS(`ROOTS`) -> `REACHED`; `DEAD_RT = DEFINED − REACHED`. For each `DEAD_RT` symbol, confirm zero `call` edges from any `REACHED` fn AND zero address-taken reloc from a reachable dispatch table.
4. **Triage.** Partition `DEAD_LINKER ∪ DEAD_RT` into whole-file-dead (no surviving symbol in the file) vs partial-file-dead (file has both live and dead symbols).
5. **Move to attic (mirror the tree).** Whole-file-dead -> `git mv src/<path> src/attic/<path>`, drop from the Makefile. Partial-file-dead -> excise the dead functions, append them to `src/attic/<path>/<file>` (create the mirror file if absent) with a provenance header (origin path + commit + the reachability reason), delete from the live file. Bomb-tombstone (keep the symbol, loud-fail body) ONLY when a still-emitted call-site references a symbol whose body must die — otherwise full-remove.
6. **GATE (the net).** After EACH removal batch, the FULL suite (smoke / pat-rung HARD / fence / corpus, ALL languages) must be NON-DECREASING. Removal is reachability-PROVEN behavior-neutral, so ANY gate drop == a root was missed (an indirectly-reached fn) -> restore it, add its entry point to `ROOTS`, re-run. This converts "is it dead?" from judgment into a linker+gate PROOF.
7. **Fixpoint.** Removing dead code makes MORE code dead (callees of removed fns). Re-run 2-6 until `gc.log` is empty AND `DEAD_RT = ∅`.
8. **COMPLETION TEST.** `--gc-sections --print-gc-sections` removes nothing from scrip · `DEAD_RT = ∅` for libscrip_rt · the `src/attic/` mirror is excluded from the build · every gate green.
