## Ō¢Č NRG ŌĆö Native Register Codegen ("beat the interpreter on statements")

**THESIS:** SPITBOL is indirect-threaded code (not fetch-decode-execute) ŌĆö SCRIP can't win by removing a dispatch loop that threading already removed. - [ ] **NRG-C ŌĆö INL: leaf-function inlining.** **INL-0** inlinability analysis in lower (ŌēżN statements, non-recursive, statically resolved ŌĆö the existing `__proc[]` table already proves the static-call set). **INL-1** splice the callee's four-port graph at the call site with paramŌåÆarg substitution. `R=INC(R)` ŌåÆ `R=R+1` inline.
